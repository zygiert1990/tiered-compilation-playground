## Quick start
1. Print disassembled bytecode: `javap -c -p target/classes/org/zygiert/Main.class`
2. Print interpreter generation time: `java -Xlog:startuptime -jar target/app.jar 10`
3. Print native code generated by interpreter: `java -XX:+UnlockDiagnosticVMOptions -XX:+PrintInterpreter -jar target/app.jar 10`
4. Print compilation and force stop compilation on given Tier: `java -XX:+PrintCompilation -XX:TieredStopAtLevel=1 -jar target/app.jar 15000`
5. Print compilation on C1: `java -XX:+UnlockDiagnosticVMOptions -XX:+PrintAssembly -XX:TieredStopAtLevel=1 -jar target/app.jar 15000`
6. Print compilation on C2: `java -XX:+UnlockDiagnosticVMOptions -XX:+PrintAssembly -XX:-BackgroundCompilation -jar target/app.jar 15000`
